HTML의 `<select>` 태그안에서 쓰이는 `<option>` 태그는 HTML과 browser에서 담당하는 친구가 아니라
사용되는 OS에서 그려주는 친구이기 때문에 CSS로 스타일링이 불가능하다.

![`<option>`태그는 os에서 담당한다.](./1.png)

그래서 예쁜 `Select` 컴포넌트를 만들고 싶다면 HTML 태그들을 이용하지 않고
직접 `Select` 컴포넌트처럼 보이게 코드를 짜고, 스타일링을 해주면 된다.

```html
<div class="selectBox">
  <button class="label">fruits 🍊</button>
  <ul class="optionList">
    <li class="optionItem">apple</li>
    <li class="optionItem">orange</li>
    <li class="optionItem">grape</li>
    <li class="optionItem">melon</li>
  </ul>
</div>
```
